# 📘 CSE0001 – Digital Literacy Repository

> 🚀 A comprehensive collection of assignments, reports, and practical tasks for the **CSE0001 Digital Literacy** course.

---

## 📌 📖 About the Project

This repository contains structured coursework and practical implementations related to **Digital Literacy**, focusing on essential digital skills required by modern students. The project covers topics such as:

- 💻 Digital tools and platforms  
- 📊 Productivity software (Docs, Sheets, Presentations)  
- 🌐 Online collaboration tools  
- 📧 Email and communication etiquette  
- 🔐 Cybersecurity and digital safety  
- 📱 Responsible social media usage  

The goal of this repository is to **build a strong foundation in digital competence**, enabling students to safely and effectively navigate the digital world.

---

## 🎯 🎓 Course Objectives

- Develop essential digital and technical skills  
- Understand professional communication (emails & social media)  
- Learn collaboration using modern platforms  
- Gain awareness of cybercrime and online safety  
- Build a digital portfolio using platforms like GitHub  

---

## 🧠 📚 Key Topics Covered

### 📊 1. Digital Tools & Productivity
- Microsoft PowerPoint, Google Slides, Canva  
- Excel & Google Sheets (formulas, charts, data analysis)

### 🌐 2. Collaboration Platforms
- Google Docs, Sheets, Forms  
- GitHub for version control and project sharing  
- Coding platforms like HackerRank, CodeChef  

### 👤 3. Digital Portfolio
- Creating professional profiles on:
  - GitHub  
  - LinkedIn  
  - Kaggle  

### 📧 4. Email & Communication Etiquette
- Writing professional emails  
- Proper salutations and tone  
- Structuring formal communication  

### 🔐 5. Cybercrime & Digital Safety
- Types of cybercrime:
  - Phishing  
  - UPI Fraud  
  - Identity Theft  
  - Cyberbullying  
- Prevention strategies  
- Reporting mechanisms  

### 📱 6. Social Media Responsibility
- Do’s and Don’ts of social media  
- Digital footprint awareness  
- Online reputation management
